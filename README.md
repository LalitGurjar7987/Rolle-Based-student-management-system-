# 🎓 Role-Based Academic Management System

## 📌 Overview

This is a web-based Academic Management System developed using Java, JSP, Servlets, and MySQL. The system supports multiple roles: Admin, Teacher, and Student, each with specific functionalities.

---

## 🚀 Features

### 👨‍💼 Admin

* Manage Students, Teachers, and Subjects

### 👨‍🏫 Teacher

* View assigned subjects
* Mark attendance
* Enter student marks

### 👨‍🎓 Student

* View subjects
* Check attendance
* View & download marksheet

---

## 🏗️ Architecture

* MVC (Model-View-Controller)
* DAO design pattern

---

## 🛠️ Technologies Used

* Java (Servlets)
* JSP (with EL)
* JDBC
* MySQL
* Apache Tomcat

---

## 🔐 Key Concepts

* Role-Based Access Control (RBAC)
* Session Management
* SQL JOIN for marksheet generation
* PreparedStatement (SQL Injection prevention)

---

## 📂 Project Structure

* Controller → Servlets
* Model → DAO classes
* View → JSP pages

---

## ▶️ How to Run

1. Import project in Eclipse
2. Configure Apache Tomcat server
3. Set up MySQL database
4. Run project on server

---

## 📊 Database

* Tables: users, students, teachers, subjects, attendance, marks
